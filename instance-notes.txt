# Noah server
=============
A couple of invocations here based on size.
# Natty 32-bit m1.small
=======================
`knife ec2 server create -r "recipe[noah::server]" -I ami-e2af508b -f m1.small -S rundeck-demo -G default,rundeck-demo -i .chef/rundeck-demo.pem -x ubuntu -d ubuntu11.04-gems -N noah`
# Natty 64-bit m1.large
=======================
`knife ec2 server create -r "recipe[noah::server]" -I ami-68ad5201 -f m1.large -S rundeck-demo -G default,rundeck-demo -i .chef/rundeck-demo.pem -x ubuntu -d ubuntu11.04-gems -N noah`
